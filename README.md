# Ted's Dragonborn Player Skins

 A collection of player skins made for 'Hideous Destructor'.

 Currently includes:

- 4 Cygnis player skins (Base, Asbestosuit, Powersuit, Draughsuit)
- Pirate Doomguy from Pirate Doom
